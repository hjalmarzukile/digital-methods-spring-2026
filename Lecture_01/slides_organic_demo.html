Lecture flow 03
Start With The End Project

Digital mixed methods project

The course points toward a 25-page project built around digital data, a self-chosen research question, and both qualitative and quantitative analysis.

Core idea

  • Collect and use digital data to answer a self-chosen question.
  • Work across qual and quant analysis rather than treating them as separate tracks.

Exam focus

  • Research design
  • Netnography
  • Computer-assisted qualitative text analysis
  • Simple quantitative analysis

Exam Hand-In Must Haves

What must be present

  • Description of the datasite informed by netnography
  • Description of the data collection
  • Description of the research design
  • Qualitative text analysis
  • Quantitative text analysis
  • Computational classification validated with a manually coded test set
