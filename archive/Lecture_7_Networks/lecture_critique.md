## Content Summary

This lecture covers: types of social ties, network structures, potentials and problems of digital network data (low-cost/low-validity, whole networks, actual interaction/ephemeral), mixed approaches to networks (thick description, network effects and change, organisational embedding), how to mix qual and quant in network research, and visual network analysis (VNA) using Gephi.

## Why It Was Removed (Likely)

The course description mentions "exploratory network analysis" as a learning outcome, but the current syllabus has dropped networks entirely — possibly due to time constraints or a pedagogical decision to focus on text analysis.

## Recommendation

The removal of networks is a significant gap given the course's stated learning outcomes. The current course description explicitly includes "exploratory network analysis" as a method students will practise. Even a half-lecture on visual network analysis would close this gap:

- **Use for representing your datasite** (Lecture 3–4: this is where I was doing fieldwork)
- **Use for sampling** (Lecture 5: snowball sampling, sampling across clusters)
- **Use for mapping social structure** (Lecture 7–8: contextualising qualitative findings)

The mixed approaches section (building networks from text data, using qual to interpret network structure) is directly relevant to the course's text analysis pipeline and would strengthen integration.

Consider at minimum adding a 30-minute session on VNA in Lecture 4 (data collection) or as an exercise class component.
